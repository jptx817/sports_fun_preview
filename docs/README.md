# Sports-Fun Dashboard Preview

This directory is generated preview content for GitHub Pages publishing.

- Source dashboard: `artifacts/site/sports_fun_dashboard_v1/`.
- Intended use: GitHub Pages preview publishing.
- Contents: static HTML, CSS, and copied JSON dashboard data.
- Private configuration should stay in the private Sports-Fun repository.

Refresh this directory from the repository root with:

```bash
./tools/publish_sports_fun_dashboard_preview.sh
```
